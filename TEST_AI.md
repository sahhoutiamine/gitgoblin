## Troubleshooting

If AI commits aren't working:

1. Check API key is set:
   ```bash
   gitgoblin enchant --show
   ```

2. Verify AI is enabled:
   ```bash
   gitgoblin enchant --enable
   ```

3. Check for error messages when running `gitgoblin sneak`

4. Ensure you have internet connection (API calls need network)

## Disabling AI Commits

If you want to go back to simple messages:
```bash
gitgoblin enchant --disable
```

## Re-enabling AI Commits

```bash
gitgoblin enchant --enable
```
